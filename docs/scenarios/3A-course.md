# 3A：课程学习

课程场景关注“能否主动回忆和应用”，而不只是生成摘要。

## 相对通用模板的变化

- 增加 Question、Example、Mistake 页面。
- 核心概念可在首次出现时建页。
- Mistake 必须来自真实错误，避免预先生成大量假易错点。
- 查询先给问题，再按需显示答案和依据。

## 使用

1. 复制 [课程规则](../../templates/scenarios/course.md) 到 `CLAUDE.md`。
2. 替换课程名称、考试范围和主要输出。
3. 运行 [课程验收案例](../../tests/cases/course.md)。

## 典型失败

- 每一段课件都拆成独立页面。
- 把模型猜测的“考试重点”写成老师明确要求。
- 没有真实答错也批量生成 Mistake。
- 复习回答只有长摘要，没有主动回忆问题。
