# 3B：学术研究

研究场景的核心不是页面数量，而是结论、方法和引用的可追溯性。

## 相对通用模板的变化

- 增加 Paper、Method、Evidence、Debate 页面。
- 一篇正式摄取的论文对应一个 Paper 页。
- Evidence 必须有页码、章节、表图或段落定位。
- 综合必须呈现反对证据和证据不足。

## 使用

1. 复制 [研究规则](../../templates/scenarios/research.md)。
2. 定义引用格式和研究问题。
3. 运行 [研究验收案例](../../tests/cases/research.md)。

## 典型失败

- 只读摘要却补写全文方法细节。
- 把作者主张当成数据结果。
- 省略与当前观点相反的论文。
- 生成不存在的 DOI、页码或参考文献。
