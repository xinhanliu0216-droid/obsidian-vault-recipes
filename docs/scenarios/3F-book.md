# 3F：图书与长材料

图书场景保留章节顺序，同时允许跨章节主题逐步浮现。

## 相对通用模板的变化

- 增加 Book、Chapter、Theme、Excerpt，可选 Character 或 Case。
- 只处理实际提供或读过的范围。
- 阅读中的 Theme 标为 tentative，完成后复核。
- 摘录必须记录页码、章节或电子位置。

## 使用

1. 复制 [图书规则](../../templates/scenarios/book.md)。
2. 定义材料类型和位置标记方法。
3. 运行 [图书验收案例](../../tests/cases/book.md)。

## 典型失败

- 没有来源却补写后续章节。
- 把一章的论点写成全书主旨。
- 摘录没有位置，无法回查。
- 为每个人名和细节创建孤立页面。
