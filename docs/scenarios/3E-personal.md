# 3E：个人成长

个人场景首先考虑隐私和措辞边界，再考虑自动整理。

## 相对通用模板的变化

- 增加 Review、Pattern、Goal、Experiment、Method。
- Pattern 至少需要两次独立记录并保留反例。
- 明确禁止医疗、心理和人格诊断。
- 私密原文默认不复制到综合页。

## 使用

1. 复制 [个人规则](../../templates/scenarios/personal.md)。
2. 决定哪些目录允许 Agent 读取。
3. 运行 [个人场景验收案例](../../tests/cases/personal.md)。

## 典型失败

- 根据一天的日记定义长期人格。
- 把情绪描述改写成客观事实。
- 在未确认时汇总私密内容。
- 用羞辱性语言评价目标执行。
