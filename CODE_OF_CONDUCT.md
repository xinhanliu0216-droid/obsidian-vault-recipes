# 社区行为准则

我们希望这里成为一个可以坦率讨论规则、案例和失败经验的协作空间。

## 期待的行为

- 围绕问题、证据和可复现结果讨论，不攻击个人。
- 尊重不同经验水平、语言习惯和工作方式。
- 提出异议时说明具体风险，并尽量给出可验证的替代方案。
- 发现自己造成误解或伤害时，及时澄清并修正。
- 保护他人的隐私，不公开 Vault 内容、身份信息或凭据。

## 不接受的行为

- 骚扰、歧视、侮辱、威胁或持续的人身攻击。
- 未经允许发布他人的私人信息或私有内容。
- 垃圾推广、冒充他人或故意干扰正常协作。
- 在被明确要求停止后继续不受欢迎的接触。

## 执行

维护者可以编辑、隐藏或删除违反准则的内容，并视情况限制参与。报告问题时请不要在公开 Issue 中披露受害者身份或敏感证据；可通过 GitHub 仓库维护者资料中的私下联系方式报告。

维护者会尽量说明处理理由，并对善意提出的申诉进行复核。
