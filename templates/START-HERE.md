# 开始使用

1. 用 Obsidian 打开当前文件夹。
2. 安装并启用 Claudian，连接你选择的本地 Agent。
3. 把第一份资料复制到 `raw/`，不要让 Agent 修改来源文件。
4. 在 Claudian 中输入：`检查这个 Vault，并为第一份资料给出摄取计划。先不要写入。`
5. 检查计划和目标文件，确认后再让 Agent 执行。

长期规则在 `CLAUDE.md`，页面模板在 `_templates/`。`index.md` 是入口，`log.md` 只追加操作记录。

9 个 LLM Wiki Skills 已放在 `.claude/skills/`，Claude Code 可以直接发现。使用 Codex 等其他 Agent 时，按对应工具的项目级 Skills 路径复制这些目录。
